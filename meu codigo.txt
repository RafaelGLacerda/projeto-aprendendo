trabalhando com uma equipe usando git

1- git pull da branch pincipal
2- gerar uma nova branch a partir da branch principal
3- trabalhar e adicionar novas funcionalidades na nova branch
4- finalizar o trabalho na branch temporaria
5- git checkout na branch principal
6- git pull 
7- fazer o git merge para unir o codigo da branch temporaria com a branch principal
8- git push da branch principal para o repositorio
